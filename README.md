Snake
====
**A recreation of the 1998 mobile phone game in Raylib.**

Features
--------
- Sprites
- Saves high score
- Settings are persistent between different sessions
- Support for different frame rates
- Upscaling options
- Sound effects

Controls
--------
- WASD to navigate menus
- Enter to select a menu option
- WASD or arrow keys to move the snake
- Escape to pause
